# About ft_Hangouts

Hangouts is an Android project focused on learning the basics of native mobile development with the Android SDK in Java.

The application allows users to create, edit, and delete contacts, exchange text messages, and store all data persistently using SQLite.

The project also introduces key Android concepts such as the Activity Lifecycle, multilingual support, responsive layouts, and UI customization.

<br>

## 🟠 Requirements

* Contacts must contain at least 5 fields.
* Use SQLite for persistence.
* Do not use Android's shared contacts table.
* No external libraries allowed.
* App icon must be the 42 logo.

<br>

## 🟠 Technologies

* Android SDK
* Java / Kotlin
* SQLite
* Android Studio

<br>

## 🟠 Features

| Feature       | Description                        |
| :------------ | :--------------------------------- |
| Contacts      | Create, edit, and delete contacts  |
| SQLite        | Persistent local database          |
| Messaging     | Send and receive messages          |
| Multilanguage | Supports two different languages   |
| Toast         | Displays last background timestamp |
| UI            | Change header color from the menu  |
| Responsive    | Supports portrait and landscape    |
